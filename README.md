# Мой Вишлист 🎁

Красивый и современный веб-сайт для публикации списка желаемых подарков. Создан с использованием HTML, CSS (Glassmorphism, CSS-анимации) и Vanilla JS. 

## Как добавлять/изменять подарки?

Все данные о подарках хранятся в простом файле `gifts.json`. 
Чтобы добавить новый подарок или изменить существующий, просто отредактируйте этот файл:

```json
[
  {
    "id": 1,
    "title": "Название подарка",
    "image": "https://ссылка-на-картинку.jpg",
    "link": "https://ссылка-на-магазин",
    "price": "1000 ₽",
    "description": "Почему я это хочу...",
    "priority": "high" // Возможные значения: high (очень хочу), medium (хорошо бы), low (подождет)
  }
]
```

## Как задеплоить на GitHub Pages?

Так как сайт полностью статический (в нем нет серверов вроде Node.js или Python, только HTML/CSS/JS/JSON), его **идеально** хостить на GitHub Pages, это абсолютно бесплатно!

1. **Создайте репозиторий на GitHub**
   Зайдите на [GitHub](https://github.com/) и создайте новый пустой репозиторий (например, `my-wishlist`). Оставьте галочки создания README и .gitignore **пустыми**.

2. **Отправьте код в GitHub**
   Откройте терминал (Terminal) в папке проекта (`/Users/kubitre/Projects/my/whishlist_gift`) и выполните команды:
   ```bash
   git init
   git add .
   git commit -m "Initial commit ✨"
   git branch -M main
   # Замените ссылку ниже на URL вашего репозитория:
   git remote add origin https://github.com/ВАШ_ЮЗЕРНЕЙМ/ВАШ_РЕПОЗИТОРИЙ.git
   git push -u origin main
   ```

3. **Включите GitHub Pages**
   - Перейдите на страницу вашего репозитория на GitHub.
   - Нажмите на вкладку **Settings** (Настройки).
   - В левом меню выберите **Pages** (в разделе "Code and automation").
   - В секции **Build and deployment**, под надписью **Source**, убедитесь, что выбрано `Deploy from a branch`.
   - В разделе **Branch** выберите `main` и папку `/(root)`.
   - Нажмите **Save**.

🎉 **Готово!** Через пару минут над этими настройками появится ссылка на ваш сайт (обычно формата `https://your-username.github.io/your-repo-name/`). Отправляйте её друзьям!
